🍽️ Mapetite: Premium Culinary Dining Platform
🌟 Project Overview
Mapetite is a modern, high-end web platform designed to help users Discover Their Next Culinary Adventure. It allows users to search for world-class restaurants globally using detailed filters and provides a seamless, premium dining discovery experience.

The core goal of the current design iteration is to establish a sophisticated, high-contrast, and trustworthy aesthetic suitable for a premium brand.

## Local frontend setup

Use Node `^20.19.0 || >=22.12.0` and pnpm 10.x.

```sh
pnpm install --frozen-lockfile
pnpm run dev
pnpm run build
pnpm run test
```

The frontend dev server defaults to `http://127.0.0.1:3000`.
